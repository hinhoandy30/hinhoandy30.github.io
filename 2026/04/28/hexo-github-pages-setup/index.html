第四步：在 GitHub 创建仓库

  1. 登录 GitHub，点右上角 + → New repository
  2. Repository name 填写 你的用户名.github.io（必须和 GitHub 用户名完全一致）
  3. 设为 Public
  4. 不要勾选 “Initialize with README”
  5. 点 Create repository

第五步：配置 SSH 密钥（一次性）

SSH 密钥让你的电脑和 GitHub 之间建立信任，之后每次发布不需要输密码。

生成密钥

查看公钥

复制输出的全部内容（以 ssh-ed25519 开头）。

添加到 GitHub

  1. GitHub 头像 → Settings → SSH and GPG keys
  2. 点 New SSH key
  3. Title 随便填（比如 MacBook）
  4. 把刚才复制的公钥粘贴进去
  5. 点 Add SSH key

测试连接

看到 Hi 你的用户名! You've successfully authenticated 就成功了。
